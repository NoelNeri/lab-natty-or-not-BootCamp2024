# Podcast: Criando Estudo Bíblico com IA Generativa 🎙️

## 📒 Descrição
Este podcast de 4 minutos oferece um diálogo entre duas teses aparentemente antagônica porém complementares. O conteúdo foi gerado por IA Generativa e revisado por humano.

## 🤖 Tecnologias Utilizadas
- IA Generativa **[ChatGPT](https://chat.openai.com)** para roteirização e revisão;
- IA Generativa **[ElevenLabs](https://www.elevenlabs.io)** para clonagem e sintetização de voz;
- Site **- [Editor de aúdio](https://www.capcut.com/editor?from_page=landing_page&__action_from=picture_V%C3%ADdeos%20profissionais%20em%20minutos,%20n%C3%A3o%20em%20horas.)** para edição e montagem do podcast.

## 🧐 Processo de Criação
"ChatGPT" foi utilizado para desenvolver o roteiro e aprimorar a narrativa do podcast. "ElevenLabs" ajudou a criar uma voz sintética envolvente para a narração, enquanto o "Capcut" foi essencial para a edição final, garantindo um fluxo suave e profissional.

## 🚀 Resultados
O episódio é uma introdução acessível e estimulante ao campo das IAs Generativas, ideal para quem busca uma compreensão rápida do assunto.

- Resultado final **[PODCAST AQUI](https://github.com/NoelNeri/prompts-for-podcast-generate-by-ia/blob/main/output/podcast_editado.MP3)**

## 💭 Reflexão
O projeto destacou a versatilidade das IAs Generativas na criação de conteúdo auditivo, abrindo novos caminhos para a produção de mídia digital.
